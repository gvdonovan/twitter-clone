## Project Setup

1. Install dependencies

```bash
npm install
```

2. Start React app

```bash
npm start
```

## Global Dependencies Required

- [Node 10+](https://node.js.org)
- [NPM 5+](https://node.js.org)
- [Git](https://git-scm.com/)
